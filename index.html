Design and Development of a Low-Cost Anthropomorphic Prosthetic Hand with Hybrid EMG and Tactile Control

  • Developed a dual-mode prosthetic robotic arm controlled via EMG signals and a foot-switch based system.
  • Designed a user-driven calibration system with EMG threshold adjustment through a web interface.
  • Integrated a self-calibration mode for personalized control tuning based on individual muscle signals.
  • Focused on improving usability, adaptability, and accessibility of prosthetic control systems.
EMG Sensors Web Interface (HTML/CSS/JS) Embedded Systems Tactile Control
View Details

This thesis focuses on the design and development of a low-cost, anthropomorphic prosthetic hand that combines EMG signal control with tactile feedback. The system is designed to be affordable and accessible for amputees in developing countries.

Key Contributions:

  • Hybrid Control System: Integration of EMG-based gesture recognition with tactile control for redundant operation.
  • Calibration Interface: Web-based EMG threshold calibration for personalized control tuning.
  • Tactile Feedback: Incorporated pressure sensors for grip force feedback.
  • EMG Control: Both three and five finger Gripping.

Edible Soft Robotics

Soft robotic system using edible and biodegradable materials for sustainable biomedical applications. Pneumatic actuation with self-oscillating acid-base reactions.

Gelatin-basedMolding & CastingPneumatic Actuation
View Details

Overview: This project explores the development of soft robotic systems using edible and biodegradable materials. The goal is to create sustainable, temporary biomedical devices that can be safely consumed or degraded after use.

Key Highlights:

  • Material Formulation: Developed gelatin-based biocompatible materials with tunable mechanical properties.
  • Pneumatic Actuation: Designed soft actuators that respond to low-pressure air input.
  • Self-Oscillating System: Implemented acid-base reaction-driven autonomous motion without external power.
  • Biomedical Applications: Potential use in assisting patients with paralysis, dysphagia, or those unable to take medication orally.

Tools & Techniques: Gelatin-based molding and casting, pneumatic control systems, chemical reaction analysis.

Non-Invasive Glucose Monitoring System

Glucose estimation using optical PPG signals with ML model for improved accuracy. Real-time prediction on embedded system.

MATLABESP32MLSignal Processing
View Details

Overview: A non-invasive glucose monitoring system that uses optical PPG (photoplethysmography) signals to estimate blood glucose levels. The system combines signal processing, machine learning, and embedded systems for real-time prediction.

Key Highlights:

  • Signal Preprocessing: Implemented noise filtering and artifact removal from PPG signals.
  • Feature Extraction: Extracted key features from PPG signals correlated with glucose levels.
  • Machine Learning Model: Developed and trained ML models to improve prediction accuracy.
  • Custom Dataset: Collected synchronized data from the developed system and a standard glucometer.
  • Real-time Deployment: Deployed the model on ESP32 for real-time glucose prediction.

Tools & Techniques: MATLAB for signal processing and ML, ESP32 for embedded deployment, PPG sensor integration.

Drug Targets in HTLV-1

In silico screening and molecular docking to identify potential drug targets in HTLV-1. Binding affinity analysis for antiviral discovery.

AutoDockPyRxBLASTGROMACS
View Details

Overview: Conducted computational analysis to identify potential drug targets in HTLV-1 (Human T-lymphotropic virus type 1) using integrated in silico approaches. The study combines bioinformatics screening, molecular docking, and dynamics simulations.

Key Highlights:

  • Bioinformatics Screening: Used BLAST, UniProt, and Clustal Omega for sequence analysis.
  • Molecular Docking: Performed docking studies using AutoDock and PyRx to evaluate protein-ligand interactions.
  • Binding Affinity Analysis: Analyzed drug-target binding affinities to identify promising candidates.
  • Molecular Dynamics: Used GROMACS for MD simulations to validate binding stability.

Tools & Techniques: AutoDock, PyRx, BLAST, Discovery Studio, GROMACS, PyMOL, Clustal Omega.
